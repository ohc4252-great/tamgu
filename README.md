# 사탐 누적 복습 마스터 (Tamgu Tracker)

사회탐구 과목(생활과 윤리, 사회문화) 학습 내용을 에빙하우스 망각 곡선 주기에 맞춰 체계적으로 복습할 수 있도록 돕는 웹 애플리케이션입니다.

## 주요 기능
- **에빙하우스 복습 로직**: 1, 3, 5, 7, 14, 30, 60, 100, 180일차 주기로 자동 알림
- **데이터 로컬 저장**: LocalStorage를 활용한 데이터 영구 보존
- **회독 시스템**: 모든 단원 완료 시 회독 수(N바퀴) 카운트 및 초기화

## 기술 스택
- HTML5, CSS3, Vanilla JavaScript
