# 사탐 전용 누적 복습 공부 기록기 (Tamgu Tracker)

## 프로젝트 개요
이 애플리케이션은 사회탐구 과목(생활과 윤리, 사회문화)의 학습 내용을 에빙하우스 망각 곡선에 기반하여 체계적으로 복습할 수 있도록 돕는 웹 애플리케이션입니다. 유저는 각 단원의 학습 시작 버튼을 눌러 복습 사이클을 시작하며, 시스템은 자동으로 복습 주기를 계산하여 "오늘의 할 일"을 관리합니다.

## 주요 기능
- **에빙하우스 기반 복습 로직**: 1, 3, 5, 7, 14, 30, 60, 100, 180일차 주기로 복습 알림 제공.
- **데이터 관리**: 브라우저 `LocalStorage`를 사용하여 학습 데이터 및 회독(N바퀴) 수 영구 보존.
- **UI/UX**:
    - **좌측 패널**: 오늘 수행해야 할 복습 목록 (체크박스 형태).
    - **중앙-우측 패널**: 과목별 아코디언 UI 및 단원 시작 버튼.
- **회독 시스템**: 모든 단원(37개)의 180일 복습이 완료되면 전체 초기화 및 회독 수 증가.

## 기술 스택
- HTML5, CSS3 (Modern Baseline), Vanilla JavaScript (ES Modules)
- LocalStorage API

## 상세 복습 사이클 (9단계) - **완료 기반 연쇄 방식**
각 단계는 **이전 단계가 완료된 시점**을 기준으로 다음 복습일이 결정됩니다.
- 1단계 (1일차): 시작 즉시 노출
- 2단계 (3일차): 1단계 완료일 + 2일 (완료 후 3일째 되는 날)
- 3단계 (5일차): 2단계 완료일 + 2일
- 4단계 (7일차): 3단계 완료일 + 2일
- 5단계 (14일차): 4단계 완료일 + 7일
- 6단계 (30일차): 5단계 완료일 + 16일
- 7단계 (60일차): 6단계 완료일 + 30일
- 8단계 (100일차): 7단계 완료일 + 40일
- 9단계 (180일차): 8단계 완료일 + 80일

*주의: 1단계 시작 후 완료하지 않으면, 아무리 시간이 지나도 다음 단계가 나타나지 않습니다.*

## 구현 단계
1. `index.html`: 레이아웃 및 정적 구조 정의.
2. `style.css`: 현대적이고 깔끔한 디자인 및 애니메이션 적용.
3. `main.js`: 데이터 처리 로직, 복습 주기 계산 알고리즘, UI 렌더링 엔진 구현.
